# INFORME-VENTAS-ACUMULADAS
Problema: La dificultad para monitorear el crecimiento progresivo de los ingresos y realizar comparativas temporales efectivas (mes a mes o año tras año) impedía una evaluación clara del cumplimiento de metas comerciales.


Solución: Implementación de un modelo de datos con medidas DAX avanzadas para calcular el acumulado de ventas y comparativas de periodos anteriores. Esto facilita la identificación de tendencias de crecimiento y permite una proyección financiera más precisa basada en el comportamiento histórico.
